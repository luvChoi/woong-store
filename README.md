# [STORE](http://woong-store.com)
[개인프로젝트] JSP Model2로 구현한 가전제품 쇼핑몰 개발<br>

 ### 개요
 - 6개월간의 개발자 양성과정에서 배운 지식들을 토대로 나만의 웹사이트 개발
 - 누구나 쉽게 접할 수 있는 쇼핑몰을 주제로 DB설계부터 서버배포까지 진행
 - 설제 서비스를 개발하는 입장으로 사용자 편의성을 고려하며 제작
